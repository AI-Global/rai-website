# Block: Jumbotron

`Jumbotron` is a hero display banner that highlights its content with an image and background image.

## Usage

```tsx
import { Jumbotron } from "@/blocks/Jumbotron";
```

## Fragment

The `Jumbotron` component exposes a `GraphQL` fragment `ListOfCardsFragment`.

```graphql
query MyContentfulQuery {
  query
  MyQuery {
    ...JumbotronFragment
  }
}
```
