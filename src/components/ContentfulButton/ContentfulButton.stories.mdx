# Button

`ContentfulButton` is used to render a button based on the contentful button response.

## Usage

```tsx
import { ContentfulButton } from "@/components/ContentfulButton";
```

## Props

<ArgsTable component={ContentfulButton} />

## Accessibility Notes

- `Button` variations should be chosen to have high contrast with the background.
