# Block: Content List

`ContentList` allows an editor to display paragraph form content with a sublimentary list beside it.

## Usage

```tsx
import { ContentList } from "@/blocks/ContentList";
```

## Fragment

The `ContentList` component exposes a `GraphQL` fragment `ContentListFragment`.

```graphql
query MyContentfulQuery {
  query
  MyQuery {
    ...ContentListFragment
  }
}
```
