# Link

A `Link` is used to navigate to another page on the website, or to link to an external website.

A `Link` should be considered when writing paragraph content. If you are creating a call to action, consider using a `Button` component.

## Usage

```tsx
import { Link } from "@/components/Link";
```

## Props

<ArgsTable component={Link} />

## Accessibility Notes

- The color of `Link` should be inherited from its parent component.

## Example

<Canvas>
  <Story name="Default" args={{ url: "/droids" }}>
    {(args) => <Link {...args}>Are Droids Allowed Here?</Link>}
  </Story>
</Canvas>

## External Links

External links can, and should be opened in a new tab using the `external` prop.

<Canvas>
  <Story
    name="External Links"
    args={{ url: "https://eddysims.com", external: true }}
  >
    {(args) => <Link {...args}>Are Droids Allowed Here?</Link>}
  </Story>
</Canvas>

## With Icon

Links can have an `Icon` added to them to give them more visual presence. The `Icon` can be added on the left or right hand side using the `iconOnRight` prop.

<Canvas>
  <Story
    name="With Icon"
    args={{ url: "/droids", icon: "Command", iconOnRight: true }}
  >
    {(args) => <Link {...args}>Are Droids Allowed Here?</Link>}
  </Story>
</Canvas>
