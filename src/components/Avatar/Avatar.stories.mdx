# Avatar

An `Avatar` is used to display a visual identefier for a person or user.

## Usage

```tsx
import { Avatar } from "@/components/Avatar";
```

## Props

<ArgsTable component={Avatar} />

## Example

<Canvas>
  <Story name="Default" args={{ ...mockImage }}>
    {(args) => <Avatar {...args} />}
  </Story>
</Canvas>

## No Image Available

An `Avatar` can also be displayed without an image. If this is the case, the `Avatar` will show an `Icon`.

<Canvas>
  <Story name="No Image Available" args={{ name: "Eddy Sims" }}>
    {(args) => <Avatar {...args} />}
  </Story>
</Canvas>
