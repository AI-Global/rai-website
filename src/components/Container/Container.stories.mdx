# Container

The `Container` component is used to center content horizontally within a page. This component determines the "readable width" of the website.

All content should be contained within a `Contianer` component

## Usage

```tsx
import { Container } from "@/components/Container";
```

## Adding a classname

A `Container` is often used to layout content that needs to be responsive to the screensize. To accomodate this, we allow for a `className` prop to be passed in.

**This technique (also called an escape hatch) should not be used for other components. This is a dangerous technique that can cause serious discrepancies in components.**
