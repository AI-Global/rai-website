# Block: Call To Action

`CallToAction` is an isolated piece of content that is meant to highlight a specific action.

## Usage

```tsx
import { CallToAction } from "@/blocks/CallToAction";
```

## Fragment

The `CallToAction` component exposes a `GraphQL` fragment `CallToActionFragment`.

```graphql
query MyContentfulQuery {
  query
  MyQuery {
    ...CallToActionFragment
  }
}
```
