# Block: List Of Cards

`ListOfCards` is used to display a list, or grid display of card information from `Contentful`. There are currently 2 types of cards that can be displayed.

- `ContentfulCard` using the `Card` component.
- `ContentfulMembershipLevels` using the `Membership` component.

## Usage

```tsx
import { ListOfCards } from "@/blocks/ListOfCards";
```

## Fragment

The `ListOfCards` component exposes a `GraphQL` fragment `ListOfCardsFragment`.

```graphql
query MyContentfulQuery {
  query
  MyQuery {
    ...ListOfCardsFragment
  }
}
```
