# WebsiteBanner

The `WebsiteBanner` is used to provide important information at the top of the website. It uses the [Banner](?path=/docs/components-banner--default-story) component.

It gathers its data from the `Contentful Micro Content` content model.

## Usage

```tsx
import { WebsiteBanner } from "@/components/WebsiteBanner";
```
